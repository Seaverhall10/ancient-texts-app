# Hallelujah AI — Policy Files

These files contain **every single guideline** that governs how Hallelujah AI responds.
There are NO hidden rules. What you see here is what you get.

## Files

| File | Purpose |
|------|---------|
| `bible_bound.yaml` | Core hermeneutic principles — how Scripture is interpreted |
| `claim_labels.yaml` | A/B/C/U evidence classification system |
| `response_format.yaml` | Required sections in study responses |
| `boundaries.yaml` | Explicit behavioral boundaries (replaces hidden guardrails) |

## How to Edit

1. **In the app**: Click the "Policy" button in the Hallelujah AI chat panel
2. **Directly**: Open any `.yaml` file in a text editor and save

Changes take effect on the next message you send.

## Principles

- **Scripture is the supreme authority** (Isaiah 40:8)
- **Every claim is labeled** with its evidence basis (A/B/C/U)
- **Nothing is hidden** — you control every guideline
- **The fear of YHWH is the beginning of wisdom** (Proverbs 9:10)
